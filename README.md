# 🏪 TokoKas - Sistem POS Toko Pribadi

**TokoKas** adalah sistem _Point of Sale_ (POS) terpadu dengan arsitektur _One Backend, Two Frontends_ yang dirancang khusus untuk pemilik toko kecil-menengah.

## 🚀 Arsitektur Sistem
- **Backend:** Node.js, Express.js, PostgreSQL (Supabase)
- **Web App (Admin & Laporan):** React.js + Vite + Tailwind CSS
- **Mobile App (Kasir Lapangan):** Flutter

## 📦 Fitur Utama
1. **Manajemen Produk:** Tambah, edit, hapus produk dengan peringatan stok kritis otomatis.
2. **Modul Kasir (POS):** Antarmuka responsif dan ramah pengguna dengan perhitungan kembalian otomatis dan dukungan berbagai metode pembayaran (Tunai, Transfer, QRIS).
3. **Manajemen Stok:** Atomic update stok pada setiap transaksi dan fitur pencatatan stok masuk dari supplier.
4. **Laporan & Analitik:** Dashboard performa harian/bulanan, grafik omzet, dan laporan pergerakan log stok produk.
